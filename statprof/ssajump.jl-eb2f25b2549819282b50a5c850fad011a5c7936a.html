267 """
268 linear_search(iterator, r)
269
270 Perform linear search for `r` over iterator. Output index j s.t. sum(array[1:j-1])
271 < r <= sum(array[1:j]), where array=collect(iterator)
272
273 Notes:
274
275 - The iterator must have all positive numbers
276 - Returns index zero if the search is unsuccessful. Assumes this corresponds to
277 the case of an infinite next reaction time and so the jump index does not
278 matter.
279 """
280 @inline function linear_search(iterator, r)
281 jidx = 0
282 idx = 1
283 for element in iterator
284 r -= element
285 if r < zero(r)
286 jidx = idx
287 break
288 end
289 idx += 1
290 end
291 return jidx
292 end
